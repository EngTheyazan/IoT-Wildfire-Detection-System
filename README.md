# IoT-Based Wildfire Detection & Management System (2025) 🌲🔥

This project presents an advanced **IoT-based solution** for early wildfire detection using **Wireless Sensor Networks (WSN)** and **Edge Machine Learning**.

## 🚀 Overview
Wildfires cause devastating environmental and economic damage. This system leverages real-time data from temperature, smoke, and gas sensors to classify fire risks at early stages using lightweight ML models (TensorFlow Lite).

## 🛠 Tech Stack & Tools
- **Simulators:** Cooja (Contiki OS), NS-3.
- **Machine Learning:** Python, TensorFlow Lite.
- **Hardware Architecture:** ESP32-based sensor nodes.
- **Networking:** Energy-efficient routing for WSN.

## 📁 Project Structure
- `/Report`: Full technical documentation.
- `/Simulation`: Scripts for Cooja and NS-3 environments.
- `/Presentation`: Project slides and visual overview.
- `/Media`: Simulation results and architecture diagrams.

## 📈 Key Results
- Improved responsiveness in detecting fire hotspots.
- High energy efficiency in data transmission.
- Low-latency classification at the edge nodes.
